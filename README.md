# TableChart
类似Excel的自定义表格控件，继承ViewGroup，代码思路借鉴于MPAndroidChart，支持缩放，横竖向滑动，合并单元格，列固定，标题固定，底部总计，自定义样式，列升序/降序，自定义点击事件等，现在只是一个demo版本，后续优化后可用于项目时候。
